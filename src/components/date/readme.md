# Date picker
Easily select dates and time, based on the [flatpickr](https://flatpickr.js.org/) library.

### Import
```javascript
import '@amber-ds/components/date';
```

### Markup
```html
<amber-date
  value="2028-12-01"
>
</amber-date>
```
