# Code
Wrapper for `<pre>` and `<code>` tags for displaying example code, with a copy to clipboard functionality.

### Import
```javascript
import '@amber-ds/components/code';
```

### Markup
```html
<amber-code-snippet>
  Content here
</amber-code-snippet>
```
