## Development

### Getting started
Clone the repo
```bash
$ git clone git@github.com:bitrockteam/amber-website.git && cd amber-website
```

Install dependencies
```bash
$ yarn
--- or ---
$ npm install
```

### Available NPM tasks
Start the project in development mode (live reload)
```bash
$ yarn start
```

Create a static build in the `./dist` folder
```bash
$ yarn build
```
