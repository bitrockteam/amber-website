# Color

The Amber color palette comes with 10 different weights, ranging from 50 (lightest) to 900 (darkest).

# Palette

Orange is the new black! Start using $orange-500 as our primary color then go up or down in weight. $orange-50 is meant to be used only as background color.

$orange-50 #FFF3E1
$orange-100 #FEDAB4
$orange-200 #FDC187
$orange-300 #FCA85A
$orange-400 #FB8F2D
$orange-500 #EB6400
$orange-600 #D55800
$orange-700 #BF4C00
$orange-800 #A94000
$orange-900 #933400
